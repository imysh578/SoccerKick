## 🎯 목적

(1) 배경
축구 경기 시 필요한 인원을 모으기 힘듦
일반인이 소속팀을 갖기 어려움
경기 매칭 시스템 기능이 있는 축구 커뮤니티의 부재

(2) 주요 고객
축구를 하고싶은 지역주민
소속팀을 갖고 싶은 일반인

(3) 개발 목표
사용자에게 높은 접근성과 편의성을 제공
축구 커뮤니티 및 매칭 기능 제공


## 🛠 사용한 기술 
- 웹 화면 구성 : `HTML` `CSS`
- DBMS : `MySQL`
- 개발 Tool : `Visual Studio Code` `github`
- AWS 배포 : `EC2` `RDS`
- 로그인 구현 : `JavaScript`
- 프레임워크 : `NodeJs` 
- 프로젝트 관리 Tool : `Google Drive` `GitHub`
